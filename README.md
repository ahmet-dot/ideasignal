# IdeaSignal

Next.js starter app for validating startup ideas with DeepSeek.

## Setup

1. Install dependencies

```bash
npm install
```

2. Create environment variable in Vercel or locally:

```bash
DEEPSEEK_API_KEY=your_key_here
```

3. Run locally

```bash
npm run dev
```

## Deploy

Push this folder to GitHub and import the repo into Vercel.
Add `DEEPSEEK_API_KEY` in Project Settings > Environment Variables.
